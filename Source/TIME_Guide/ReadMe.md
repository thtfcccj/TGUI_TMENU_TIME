## 导航输入法方案
   此文档符合MarkDown格式规范
   此输入法作为TGUI系统的一部分，使用自由软件协议。

---------------------------------------------------------------
### 总体
### 导航输入法：
   是指利用实现中英文符号输入的方法。

### 支持的输入法：
  * 大写字母
  * 小写字母
  * 数字
  * 英文符号
  * Abc输入法汉字(利用现有代码资源)

### 设计原则：
  * 以最小的按键实现想要的输入
  * 界面尽量友好
  * 多种输入法支持
  * TGUI实现，与硬件与具体应用无关

### 显示屏需求及说明：
  显示屏最小为240*128，即30字符，8行， 这里需要6行。
  需要的6行显示， 从上到下分为：
  * 字符输入区：1行
  * 横线间隔：  1行
  * 输入提示区：1行
  * 导航提示区：3行

### 按键需求及说明：
  * ↑: 上键
  * ↓: 下键
  * ←: 左键
  * →: 右键
  * 确认: 确认键，在导航提示区显示为“●”
  * 退出: 退出键，在导航提示区显示为“○”

---------------------------------------------------------------
### 字符编辑模式
  选择字符后进入此模式：此时显示为：
    ┌─────────────── ┐
    │　输入设备| 　　　　　　　　　 |    表示光标
    ├─────────────── | 此行显示，C形表示边界
    │编辑模式: 　　　　　　
    │　　　　插入↑●保存退出
    │　　　左移←  →右移　 
    │　　　　删除↓ ○直接退出
    └───────────────
   这里按↓把“备”字删除了

当用户按直接退出键时，为避免用户误操作，给出提示：
    ┌─────────────── ┐
    │　输入设备| 　　　　　　　　　 | 
    ├─────────────── |
    │不保存退出吗? 　　　　　　
    │　　　　    ↑●确认不保存退出
    │　　　    ←  → 　 
    │　　　　    ↓ ○返回编辑模式
    └───────────────

### 输入法选择模式:
  字符编辑模式按插入：此时显示为：
    ┌───────────────
    │　输入| 　　　　　　　　　 
    ├───────────────
    │输入法选择: 　　　　　　
    │　　　　   A↑ ●拼音
    │　　    ,.←  →0-9　 
    │　　　　   a↓ ○返回
    └───────────────
---------------------------------------------------------------
### 大小写字母输入法模式:
  以大写输入法为例，输入法选择模式按↑, 此时显示为：
    ┌───────────────
    │　输入| 　　　　　　　　　 
    ├───────────────
    │A: 选择分组　　　　　　
    │　　　 ABCDE↑ ●KLMNO
    │　　 FGHIJ←  →PQRST 
    │　　  UVWXYZ↓ ○返回
    └───────────────
  如想输入J,按←后：
    ┌───────────────
    │　输入| 　　　　　　　　　 
    ├───────────────
    │A: 选择字母　　　　　　
    │　　　     F↑ ●H
    │　　     G←  →I 
    │　　       J↓ ○返回
    └───────────────
  按↓后输入完成，插入输入的字符后，重新进入当前输入模式

  特别地，当想输入Y或Z时，在选择分组点↓后提示
    ┌───────────────
    │　输入J| 　　　　　　　　　 
    ├───────────────
    │A: 选择字母(z:快速↓两次)　　　　　　
    │　　　     U↑ ●W
    │　　     V←  →X 
    │　　      YZ↓ ○返回
    └───────────────
  此时，按一下↓选Y, 快速按两下↓选Z

### 符号输入法模式:
  输入法选择模式按←：此时显示为：
    ┌───────────────
    │　输入J| 　　　　　　　　　 
    ├───────────────
    │,.: 选择分组　　　　　　
    │　　   .[~],↑ ●下一组
    │　　 "(:)'←  → /<->|
    │　　   !{#}@↓ ○返回
    └───────────────
    若想输入的符号不在本组，按●继续选择
    假定要输入[，按↑键后：
    ┌───────────────
    │　输入J| 　　　　　　　　　 
    ├───────────────
    │,.: 选择符号　　　　　　
    │　　　     .↑ ●~
    │　　     [←  →] 
    │　　       ,↓ ○返回
    └───────────────
  按确定键后输入完成，退回编辑模式:
    ┌───────────────
    │　输入J~| 　　　　　　　　　
    ├───────────────
    │编辑模式: 　　　　　　
    │　　　　插入↑ ●保存退出
    │　　　左移←  →右移　 
    │　　　　删除↓ ○直接退出
    └───────────────

### 数字输入法模式:
  输入法选择模式按→：此时显示为：
    ┌───────────────
    │　输入J~| 　　　　　　　　　 
    ├───────────────
    │0-9: (快速按两次选择第二个)　　　　　　　
    │　　     01↑ ●45
    │　　   23←  →67
    │　　     89↓ ○返回
    └───────────────
    假定要输入1，快带按两次↑键后退回编辑模式:
    ┌───────────────
    │　输入J~1| 　　　　　　　　　 
    ├───────────────
    │编辑模式: 　　　　　　
    │　　　　插入↑ ●保存退出
    │　　　左移←  →右移　 
    │　　　　删除↓ ○直接退出
    └───────────────

 ### abc拼音输入法模式:
  输入法选择模式按●, 此时显示为：
    ┌───────────────
    │　输入J~1| 　　　　　　　　　 
    ├───────────────
    │拼音: 快速按两次为第二组        第二屏：全部完成仍未显示直接●选择　　　　　　　
    │　abc def  ↑ ●拼音完成
    │　ghi jkl←  → mno pqrs
    │　tuv wxyz ↓ ○返回
    └───────────────
  假定想输入 “家”，按两次←选择jkl后：
    ┌───────────────
    │　输入J~1| 　　　　　　　　　 
    ├───────────────
    │ji ka li jie liu　　　　　　
    │　abc def  ↑ ●拼音完成
    │　ghi jkl←  → mno pqrs
    │　tuv wxyz ↓ ○删除
    └───────────────
  拼音jia没有出现，按1次←选择ghi后：
    ┌───────────────
    │　输入J~1| 　　　　　　　　　 
    ├───────────────
    │ji jia li liu lin liu l...(显示不下时...提示，若输入完成还没则直接确认)　
    │　abc def  ↑ ●拼音完成
    │　ghi jkl←  → mno pqrs
    │　tuv wxyz ↓ ○删除
    └───────────────
  拼音jia出现了，按确定键选择拼音：
    ┌───────────────
    │　输入J~1| 　　　　　　　　　 
    ├───────────────
    │拼音: 拼音选择　　　　　　
    │　       ji↑ ●下一组
    │　   jia ←  → liu
    │　      lin↓ ○返回
    └───────────────
  按←键确认为jia，提示:
    ┌───────────────
    │　输入J~1| 　　　　　　　　　 
    ├───────────────
    │拼音: 汉字选择　　　　　　
    │　       加↑ ●下一组
    │　    架 ←  → 家
    │　       价↓ ○返回
    └───────────────
  家已经在本页了(否则●查看下一组，可回环)
  按→选择"家"后，返回编辑模式:
    ┌───────────────
    │　输入J~1| 　　　　　　　　　 
    ├───────────────
    │编辑模式: 　　　　　　
    │　　　　插入↑ ●保存退出
    │　　　左移←  →右移　 
    │　　　　删除↓ ○直接退出
    └───────────────
